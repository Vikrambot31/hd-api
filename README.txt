==========================================================
  Human Design Calculator — Версия для веб-сайта
==========================================================

Что находится в этой папке:
  calculator_web.html  — Главный файл калькулятора (открыть в браузере)
  fullhd_data.js       — Таблица FullHD (13 824 записи врат/линий из Excel)
  hd_engine.js         — JavaScript-движок расчётов (алгоритм Moshier)
  README.txt           — Этот файл

----------------------------------------------------------
КАК ИСПОЛЬЗОВАТЬ (вариант 1 — прямо с компьютера):
----------------------------------------------------------
ВАЖНО: Из-за политики браузеров нельзя открыть
calculator_web.html двойным кликом (файл:// блокирует JS).
Нужен локальный веб-сервер.

Способ А — Python (если установлен):
  1. Откройте папку web_calculator в терминале/PowerShell
  2. Введите команду:
     python -m http.server 8000
  3. Откройте в браузере: http://localhost:8000/calculator_web.html
  4. Для остановки: Ctrl+C

Способ Б — Node.js (если установлен):
  npx serve .
  Потом открыть указанный адрес.

----------------------------------------------------------
КАК РАЗМЕСТИТЬ НА САЙТЕ (Netlify, GitHub Pages, и т.д.):
----------------------------------------------------------
1. Создайте папку на вашем хостинге
2. Загрузите ВСЕ ТРИ файла:
   - calculator_web.html
   - fullhd_data.js
   - hd_engine.js
3. Откройте calculator_web.html через URL сайта

Пример для Netlify (vikramhd2027.netlify.app):
1. Зайдите на netlify.com → ваш сайт
2. Deploys → перетащите папку web_calculator
3. Готово! Калькулятор доступен по адресу:
   https://vikramhd2027.netlify.app/calculator_web.html

----------------------------------------------------------
ТОЧНОСТЬ:
----------------------------------------------------------
Расчёт использует формулы Moshier (Jean Meeus,
"Astronomical Algorithms"). Таблица FullHD — из файла
!Рассчеты_upd_v.5.xlsx. Точность: ±0.003° для Солнца,
±0.01° для Луны, ±0.1° для дальних планет.

Проверено на эталоне 31.01.1984 07:25 UTC+3:
  Тип: Рефлектор, Профиль: 4/6, Дата дизайна: 05.11.1983

----------------------------------------------------------
ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:
----------------------------------------------------------
- Любой современный браузер (Chrome, Firefox, Edge, Safari)
- JavaScript должен быть включён
- Не требует Python, Node.js, или любых других программ
  при использовании на сайте
